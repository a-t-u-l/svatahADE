Svatah ADE

ADE is an acronym for Automation Development Environment. It is a desktop client built using Electron and Svatah's behaviour driven testing framework to help you build and run your Svatah Flows locally. It provides you with one click setup for running your integration test for web apps and API. It is super easy to create a new project, just follow the getting started guides to create your project.

API Client

Ever wondered how your API's services will respond to user requests? You can test your API's locally using the client.

View Automation Projects

Collection of all your test projects are stored here for you to view, modify or execute.

New Project

Svatah works on behavior driven model where you can write user stories in plain english using Svatah's vocabulary and connect them together as flows.

View Results

All your test results are stored locally in descending order of their execution. Your results are captured step by step basis giving you a clear insight on your usr journey.
